How to create a website like this one

Posted on June 24, 2018 in Blog

Flex theme is availabe online and I use Pelican to generate this static site and host it on Github. There are a few steps 1 Create webpages using Pelican 2 upload your files to github and setup github page. - You can upload it by set the output filder as the rootfile of your github repository - But a convenient tool to commit a whole a specific filder to github is ghp- import, which allow you to have your .git outside of the output folder - I use Windows, so I have created a bat file to compile and push the file to the repository. I use the About page as my customized homepage so I just copy the about.html to replace the index.html 

pelican
copy "output\pages\about.html" "output\index.html" /y
ghp-import output -b master
git push origin master
